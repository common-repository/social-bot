A plugin to create response for Facebook Messenger bot

== Description ==

Social Bot is a powerful WordPress plugin that enables you to effortlessly create and manage chatbots for Facebook Messenger, enhancing user engagement and interaction on your website.

### Key Features

- **User-Friendly Interface:** Easily create and manage multiple chatbots through an intuitive and user-friendly interface.
  
- **Seamless Integration:** Link your Facebook Messenger bots with specific posts on your WordPress site, providing a personalized and cohesive user experience.

### Social Bot Settings

- **Enable Facebook Messenger Bot:** Unlock additional options by activating the Facebook Messenger Bot feature.

- **Development Settings:** Facilitate testing with options such as "Debug by var_dump," particularly useful for developers leveraging tools like Postman.

### Getting Started

1. Create your app on the [Facebook Developer Control Panel](https://developers.facebook.com/apps/create/).

2. Obtain the necessary credentials and effortlessly link your posts to the Messenger bot created with this plugin.

### Why Choose Social Bot?

- **Customizable:** Tailor your chatbot's behavior to suit your website's unique requirements.

- **Robust Debugging:** Enable development settings for comprehensive debugging, including var_dump support.

- **Regular Updates:** Stay current with our frequent updates, ensuring compatibility with the latest WordPress versions.
